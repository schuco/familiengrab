# Familiengrab Huckarde – GitHub Pages

Statische Seite für das Familiengrab (Katholischer Friedhof, Huckarde) mit Personenliste, Fotos und Verwandtschafts-SVG.

## Nutzung
- `index.html` und `media/`-Ordner ins Repo legen.
- GitHub Pages im Repo aktivieren (Branch `main`, Ordner `/`).
- `gvexport-1.svg` bleibt im Root und wird eingebettet angezeigt.

## Entfernte Hinweise (aus removed-notes.txt)
- Datenpflege erfolgt lokal; diese Seite zeigt eine veröffentlichbare Zusammenstellung.
- Fotos/Quellen liegen lokal im Ordner `media/` (Dateinamen siehe Karten).
- GitHub Pages: `index.html` ins Root legen und Pages auf `main` oder `docs/` stellen.
